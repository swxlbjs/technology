tcs

部署

前置条件

当前版本安装分为3+x/6+x，3个或6个管理节点，x个业务节点

  • TCS版本：TCS-2.1.2
  • 系统：tlinux 2.4
  • 内核：4.14
  • Master： 16C/32G/100G/500G*3
  • Worker： 16C/32G/100G/500G*2
  • vip：4个

安装步骤

  • 选其中一个master当作安装节点下载tcs 
    mkfs.xfs -n ftype=1 /dev/vdb -f
echo "`blkid | grep /dev/vdb | awk '{print $2}'` /data xfs defaults,noatime,nodiratime,prjquota 0 2" >> /etc/fstab
mount -a
mkdir -p /data/tce_dc
yum -y install screen
screen -R tcs -L
cd /data/tce_dc
wget "tcs下载地址"
tar xvf /data/tce_dc/tcs.tar.gz -C /data/tce_dc/ --strip-components 2
  • 打开tcs-planning生成配置文件conf.zip 
    主机配置：按照要求填写ip、端口、密码、磁盘
参数配置： 环境名(示例：tcs)、主域名（示例：auto-deploy.tcs.fsphere.cn）、svc-lb-vip需要更改，pod_cidr_pool、service_cidr_pool根据"查看帮助?"修改
将conf.zip上传至/data/tce_dc
  • 执行安装(整个过程大约持续4个小时) 
    sh /data/tce_dc/tcs/deploy.sh

附加备注

  • HA-vip：https://console.cloud.tencent.com/vpc/havip
  • 本地隧道：ssh -o stricthostkeychecking=no -N -f -D 0.0.0.0:1080 root@服务器公网ip -p 服务器公网端口
